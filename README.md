## 📱 О проекте

**Android FRP Activity Launcher** — это мощный веб-интерфейс, предназначенный для запуска скрытых активностей и системных настроек на устройствах Android. Проект создан для технических специалистов, тестировщиков и энтузиастов, работающих с FRP-блокировками, диагностикой и настройкой мобильных устройств.

Инструмент не требует установки, не содержит JavaScript (фильтрация работает на чистом CSS) и полностью адаптирован для мобильных устройств. Все ссылки реализованы через `intent://`-схемы, что позволяет напрямую обращаться к системным приложениям и скрытым меню.

> ⚠️ **Важно:** Проект предназначен **только для законного тестирования**, отладки и образовательных целей. Использование в обход защиты устройств, которые вам не принадлежат, может нарушать законодательство.

## 🚀 Быстрый старт

### Онлайн-версия (GitHub Pages)

Просто перейдите по ссылке и используйте инструмент прямо в браузере вашего Android-устройства:

👉 **[https://ваш-username.github.io/ваш-репозиторий/index.html](https://ваш-username.github.io/ваш-репозиторий/index.html)**  
*(замените `ваш-username` и `ваш-репозиторий` на актуальные)*

### Локальный запуск

1. Скачайте файл `index.html` из репозитория.
2. Перешлите его на Android-устройство (через USB, облако, мессенджер).
3. Откройте файл любым браузером (Chrome, Firefox, Samsung Internet).
4. Нажмите на любую ссылку — откроется соответствующая активность Android.

> ✅ Для корректной работы ссылок браузер должен поддерживать обработку `intent://`. Современные браузеры (Chrome, Edge, Brave) делают это автоматически.

## 🕹️ Как пользоваться

1. **Выбор категории** — нажмите на кнопку фильтра (Все / Общие / Google / Samsung и т.д.), чтобы отобразить только нужный раздел.
2. **Запуск активности** — просто тапните по любой строке. Откроется системное приложение, мастер настройки или скрытое меню (если доступно на вашем устройстве).
3. **Поиск** — используйте поиск в браузере (Ctrl+F / «Найти на странице») для быстрого перехода к нужному пункту.

### Примеры ссылок, которые можно запустить:

| Действие | INTENT |
|----------|--------|
| Открыть настройки | `intent://com.android.settings/#Intent;scheme=android-app;end` |
| Набор номера | `tel:123456789` |
| Калькулятор AOSP | `intent://com.android.calculator2/#Intent;scheme=android-app;end` |
| Samsung Secure Folder | `intent://com.samsung.android.knox.securefolder/#Intent;scheme=android-app;end` |
| Xiaomi Security Center | `intent://com.miui.securitycenter/#Intent;scheme=android-app;end` |

## 🗂️ Структура разделов

- **Samsung / Xiaomi / OnePlus / OPPO / Huawei / Motorola** — вендорские приложения и сервисные меню.
- **LG / Sony / HTC / Nokia / ASUS / Realme / Infinix / Pixel** — уникальные активности для каждой марки.

## 🛠️ Технические детали

- **Чистый HTML5 / CSS3** — нулевая зависимость от внешних библиотек.
- **CSS-фильтрация** — переключение категорий реализовано через `radio` + селекторы `:checked`, без использования JavaScript.
- **Адаптивная вёрстка** — `viewport`, медиа-запросы, гибкие сетки.
- **Тёмная тема** — поддержка `prefers-color-scheme: dark`.
- **Ссылки INTENT** — соответствуют синтаксису Android для прямого вызова активностей.

## 📥 Установка на GitHub Pages

1. Форкните этот репозиторий.
2. В настройках репозитория (`Settings` → `Pages`) выберите ветку `main` и папку `/ (root)`.
3. Сохраните — через минуту сайт будет доступен по адресу:  
   `https://[ваш-username].github.io/[репозиторий]/`
4. Убедитесь, что файл `index.html` лежит в корне.

## ⚠️ Ограничения и рекомендации

- **Не все активности доступны на всех прошивках** — некоторые INTENT-схемы могут быть заблокированы производителем.
- **Android 11+** требует дополнительных разрешений для некоторых активностей (например, `QUERY_ALL_PACKAGES`). В большинстве случаев браузер всё равно может открыть приложение, но может показать предупреждение.
- Для работы **Smart Lock** и **Google Trust Agent** необходимо, чтобы на устройстве были включены сервисы Google Play.
- Секретные коды (`*#0*#`, `*#*#4636#*#*` и т.п.) открываются через стандартный набор номера.
