ℹ️ О приложении

FRP Activity Launcher Pro

Версия 3.0.0 (Build 2026.05) · Лицензия MIT

📱 Описание

Мощный веб-лаунчер INTENT-активностей для Android-устройств. Позволяет запускать системные настройки, сервисные меню, скрытые активности и приложения через intent://-схемы. Идеальный инструмент для технических специалистов, тестировщиков и энтузиастов, работающих с FRP-блокировками и диагностикой.

✨ Основные возможности

  • 🚀 Запуск любых INTENT-активностей Android
  • 📱 Поддержка 20+ брендов: Samsung, Xiaomi, Google Pixel, OnePlus, OPPO, Huawei, Motorola, LG, Sony, HTC, Nokia, ASUS, Realme, Infinix и другие
  • ⭐ Избранное — сохраняйте часто используемые активности
  • 🕐 История — отслеживайте последние 50 запусков
  • 🔍 Умный поиск по всем активностям
  • 🌗 Тёмная / светлая тема с автоматическим переключением
  • 📱 Адаптивный интерфейс под любые экраны
  • 💾 Экспорт / импорт данных (избранное + история) в JSON
  • 🔧 Сервисные коды и диагностика устройства
  • 🎨 Material Design 3 интерфейс с плавными анимациями

🛠️ Технические детали

  • Чистый HTML5 / CSS3 / минимальный JavaScript (только для localStorage)
  • CSS-фильтрация через :checked и :target
  • Боковое меню (drawer) с навигацией по отдельным экранам
  • Аккордеоны для компактного отображения категорий
  • Ripple-эффект и тактильная обратная связь при нажатии
  • Анимации с возможностью отключения для слабых устройств
  • Полностью статический сайт — работает на GitHub Pages без сервера

📋 Структура разделов

  • Общие инструменты Android — настройки, USB, ADB, камера, контакты, календарь
  • Экран блокировки и безопасность — Smart Lock, отпечатки, биометрия, PIN-код SIM
  • Системные настройки — экран, энергосбережение, Wi-Fi, Bluetooth, NFC
  • Сторонние приложения — HushSMS, Alliance Shield, Activity Launcher, Test DPC, ES Проводник
  • Google сервисы — Chrome, Gmail, YouTube, Карты, Play Store, Drive, Meet
  • Вендорские разделы — Samsung, Xiaomi, OnePlus, OPPO, Huawei, Motorola, LG, Sony, HTC, Nokia, ASUS, Realme, Infinix, Google Pixel

⚠️ Важное предупреждение

Приложение предназначено только для законного тестирования, отладки и образовательных целей. Использование в обход защиты устройств, которые вам не принадлежат, может нарушать законодательство. Некоторые INTENT-активности могут быть недоступны на определённых прошивках или версиях Android. Для работы секретных кодов (tel:*) требуется стандартный набор номера.
